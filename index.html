My research

My masters' project is about optimization of pump spectral shape for quantum light generation. The code used for this project can be found on my GitHub page. Using the power of XLA from JAX, a machine learning module developped by Google, fast computation of iterative optimization schemes are made to minimize spectral entanglement between light generated by spontenous parametric down-conversion.
